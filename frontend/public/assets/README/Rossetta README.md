Rossetta: 경계선 지능 아동 학습 지원

Rossetta는 지역 간 교육 불평등으로 인한 학습을 도와주고, 경계선 지능 아동들의 능력을 향상 시키는 것을 목표로 합니다.
## 문제 설명
"한국의 경계선 지능 현황 or 지역 간 불평등"
경계선 지능 장애는 성인이 되어서 취업 등 일상생활의 불편함, 사회성 부족 등등 여러 문제를 불러 일으킨다.
